Configure Postman

Set up Postman for testing Stripe API endpoints.

You must have a Stripe secret API key before configuring Postman.

  1. Download and install Postman.
    Navigate to https://www.postman.com/downloads/.
  2. Store your secret key securely.
    1. Go to Vault on the bottom menu.
    2. Enter a Key name, for example STRIPE_SECRET_KEY.
    3. Enter the Stripe secret API key in Value.
    4. Leave Allowed domains blank.
    Your secret key is now stored securely and can be referenced in the format {{vault:KEY_NAME}}, where KEY_NAME is your Key name. For example, {{vault:STRIPE_SECRET_KEY}}.
  3. Configure collection authorization.
    1. Select New and then select Collection. Rename if desired, for example, Stripe API testing.
    2. Select your collection and then select Auth.
    3. Set Type to Basic Auth.
    4. Set Username to {{vault:STRIPE_SECRET_KEY}} to access the key stored in your vault.
    5. Leave Password blank.
  4. Set the base URL variable.
    1. Select your collection and then select Variables.
    2. Enter a Variable named base_url.
    3. Set Value to https://api.stripe.com/v1.
